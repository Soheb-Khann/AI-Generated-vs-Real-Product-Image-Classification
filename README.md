# AI-Generated-vs-Real-Product-Image-Classification-
Binary image classification pipeline to detect AI-generated product listings in e-commerce. Fine-tuned EfficientNet-B3 and ConvNeXt-Base using transfer learning, CutMix augmentation, snapshot ensembling, and threshold optimisation. F1-score improved from 0.62 to 0.95 on a real-world Etsy dataset. Built with Python, PyTorch, and Scikit-learn.
